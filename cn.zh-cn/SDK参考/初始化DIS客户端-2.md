# 初始化DIS客户端<a name="dgc_06_0026"></a>

您可以使用以下方法初始化DIS SDK客户端实例。其中，“endpoint“，“ak“，“sk“，“region“，“projectId“信息请参看[获取认证信息](获取认证信息.md#dgc_06_0005)。

-   ```
cli = disclient(endpoint='**your-endpoint**',
            ak='**your-ak**',
            sk='**your-sk**',
            projectid='**your-projectid**',
            region='**your-region**')
```
