# 下载数据<a name="dis_02_0019"></a>

## 功能介绍

本接口用于从DIS通道中下载数据。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DIS&api=ConsumeRecords)中调试该接口。

## URI

GET /v2/\{project\_id\}/records
