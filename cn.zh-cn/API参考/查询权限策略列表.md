# 查询权限策略列表<a name="ListPolicies"></a>

## 功能介绍

本接口用于查询指定通道的权限策略列表。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DIS&api=ListPolicies)中调试该接口。

## URI

GET /v2/\{project\_id\}/streams/\{stream\_name\}/policies

## 请求示例

查询权限策略列表

```
GET https://{Endpoint}/v2/{project_id}/streams/{stream_name}/policies
```

## 响应示例

**状态码： 200**

正常返回

```
{
  "streamId" : "CiFdELMr0401K9GGZlp",
  "rules" : [ {
    "action_type" : "putRecords",
    "principal" : "3b3f237122574xxxxb74482ae11005ba.*",
    "principal_name" : "anotherusername",
    "effect" : "accept"
  } ]
}
```
