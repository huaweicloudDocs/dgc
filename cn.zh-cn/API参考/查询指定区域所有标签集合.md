# 查询指定区域所有标签集合<a name="ListTags"></a>

## 功能介绍

该接口用于查询指定区域所有标签集合。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DIS&api=ListTags)中调试该接口。

## URI

GET /v2/\{project\_id\}/stream/tags

## 请求示例

查询指定区域所有标签集合。

```
GET https://{Endpoint}/v2/{project_id}/stream/tags
```

## 响应示例

**状态码： 200**

标签集合响应体。

```
{
  "tags" : [ {
    "key" : "key1",
    "values" : [ "value1", "value2" ]
  }, {
    "key" : "key2",
    "values" : [ "value1", "value2" ]
  } ]
}
```
