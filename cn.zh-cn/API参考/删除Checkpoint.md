# 删除Checkpoint<a name="DeleteCheckpoint"></a>

## 功能介绍

本接口用于删除Checkpoint。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DIS&api=DeleteCheckpoint)中调试该接口。

## URI
