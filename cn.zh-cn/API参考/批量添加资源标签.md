# 批量添加资源标签<a name="BatchCreateTags"></a>

## 功能介绍

该接口用于批量添加资源（通道等）标签。此接口为幂等接口：创建时如果请求体中存在重复key则报错。创建时，不允许设置重复key数据，如果数据库已存在该key，就覆盖value的值。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DIS&api=BatchCreateTags)中调试该接口。

## URI

POST /v2/\{project\_id\}/stream/\{stream\_id\}/tags/action

## 请求示例

批量添加资源标签

```
POST https://{Endpoint}/v2/{project_id}/stream/{stream_id}/tags/action

{
  "action" : "create",
  "tags" : [ {
    "key" : "key1",
    "value" : "value1"
  }, {
    "key" : "key2",
    "value" : "value3"
  } ]
}
```
