# 查询分区监控<a name="ShowPartitionMetrics"></a>

## 功能介绍

本接口用于查询通道指定分区的监控数据。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DIS&api=ShowPartitionMetrics)中调试该接口。

## URI

GET /v2/\{project\_id\}/streams/\{stream\_name\}/partitions/\{partition\_id\}/metrics
