# 查看通道详情<a name="dis_02_0025"></a>

## 功能介绍

本接口用于查询指定通道的详情。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DIS&api=ShowStream)中调试该接口。

## URI
