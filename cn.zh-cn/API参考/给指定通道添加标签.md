# 给指定通道添加标签<a name="CreateTag"></a>

## 功能介绍

本接口用于给指定通道添加标签。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DIS&api=CreateTag)中调试该接口。
