# 查询指定通道的标签信息<a name="ShowStreamTags"></a>

## 功能介绍

该接口用于查询指定通道的标签信息。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DIS&api=ShowStreamTags)中调试该接口。

## URI

GET /v2/\{project\_id\}/stream/\{stream\_id\}/tags

## 请求示例

查询指定通道的标签信息。

```
GET https://{Endpoint}/v2/{project_id}/stream/{stream_id}/tags
```

## 响应示例

**状态码： 200**

通道标签信息响应体。

```
{
  "tags" : [ {
    "key" : "key1",
    "value" : "value1"
  }, {
    "key" : "key2",
    "value" : "value3"
  } ]
}
```
