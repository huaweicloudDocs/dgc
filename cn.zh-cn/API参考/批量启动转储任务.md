# 批量启动转储任务<a name="BatchStartTransferTask"></a>

## 功能介绍

此接口用于批量启动转储任务。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DIS&api=BatchStartTransferTask)中调试该接口。

## URI

POST /v2/\{project\_id\}/streams/\{stream\_name\}/transfer-tasks/action

## 请求示例

```

{
  "action" : "start",
  "tasks" : [ {
    "id" : "9dSu1wfCytSk1aOLxvF"
  } ]
}
```
