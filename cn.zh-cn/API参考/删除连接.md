# 删除连接<a name="DeleteLink"></a>

## 功能介绍

删除连接接口。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CDM&api=DeleteLink)中调试该接口。

## URI

DELETE /v1.1/\{project\_id\}/clusters/\{cluster\_id\}/cdm/link/\{link\_name\}
