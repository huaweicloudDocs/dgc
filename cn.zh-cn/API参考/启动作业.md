# 启动作业<a name="StartJob"></a>

## 功能介绍

启动作业接口。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CDM&api=StartJob)中调试该接口。

## URI

PUT /v1.1/\{project\_id\}/clusters/\{cluster\_id\}/cdm/job/\{job\_name\}/start
