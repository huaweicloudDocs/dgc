# 创建消费App<a name="CreateApp"></a>

## 功能介绍

本接口用于创建消费APP。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DIS&api=CreateApp)中调试该接口。
