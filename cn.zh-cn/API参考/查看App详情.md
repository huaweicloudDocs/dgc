# 查看App详情<a name="ShowApp"></a>

## 功能介绍

本接口用于查询APP详情。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DIS&api=ShowApp)中调试该接口。
