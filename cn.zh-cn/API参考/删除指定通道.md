# 删除指定通道<a name="DeleteStream"></a>

## 功能介绍

本接口用于删除指定通道。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DIS&api=DeleteStream)中调试该接口。

## URI
